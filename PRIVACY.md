# Privacy Policy — HTML Copy Selector

_Last updated: June 26, 2026_

HTML Copy Selector does **not** collect, store, or transmit any user data.

## What the extension does

When you right-click an element and choose a "Copy HTML for AI" menu item, the
extension reads that element's HTML and writes it to your clipboard. That is the
only data it touches.

## Data handling

- **No data is collected.** The extension does not gather any personal or usage data.
- **No data is stored.** Nothing is saved to disk, local storage, or any database.
- **No data is transmitted.** Nothing is sent to the developer or any third party.
  All processing happens locally in your browser.
- **No remote code.** The extension runs only the code bundled in the package.
- **No analytics, no tracking, no ads.**

## Permissions

- `contextMenus` — to add items to the right-click menu.
- Host access (`<all_urls>`) — to read the HTML of the element you right-click so
  it can be copied to your clipboard. It is used only at the moment you invoke a
  menu item, and the result never leaves your device.
